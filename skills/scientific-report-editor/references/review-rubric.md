# Review Rubric for Scientific Report Editor

Use this file for Pass 2, Pass 4, and Pass 5 review loops.

## Scientific validity
- Validate each factual claim against supplied sources.
- Flag ungrounded causal claims and unsupported inference jumps.
- Confirm units, ranges, statistical notation, and assumptions are explicit.

## Method and evidence
- Check reproducibility details, sample framing, and limitations.
- Verify that method descriptions match claimed results and that caveats are stated.

## Writing quality
- Require dense but readable paragraph flow.
- Ensure section transitions and terminology are consistent.
- Remove hedging inflation and unsupported superlatives.

## Risks and integrity
- Identify potential overclaims, ambiguity, and hidden contradictions.
- Catch internal inconsistencies before final output.
- Mark each issue as blocking or optional.
