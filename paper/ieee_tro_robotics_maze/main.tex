\title{Deterministic Static Grid-Maze Planner Benchmarking for Reproducible Robotics Experiments}

\author{Cesar Contreras%
\thanks{Cesar Contreras is an independent robotics researcher based in California, USA.}%
}

\maketitle

\begin{abstract}
This paper presents a deterministic maze-navigation benchmark stack focusing on integration-level
novelty for reproducible robotics experiments. The implementation unifies planner execution,
output normalization, path validation, and artifact generation in a single repository workflow
built around deterministic seed propagation. In the executed benchmark snapshot, the system
evaluates 12 planners on 50 generated \(15\times15\) mazes with trial-level reporting for
success, runtime, and search expansions. Results show that within this static regime,
weighted A* achieves the lowest mean solve time (0.35 ms), though search-effort profiles
highlight non-trivial performance tradeoffs. The contribution is a traceable benchmarking
workflow for static grid-maze navigation; claims are explicitly bounded to this regime and
do not assert general superiority in dynamic or kinodynamic navigation domains.
\end{abstract}
